# 💻 Fundamentals of Computer Science Project

## 🔢 Number Systems Converter & Binary Calculator


## 📌 Overview

This project was developed as part of the **Fundamentals of Computer Science (FCS)** course.
👥 Team size: 2 students

It is a **Python console application** that focuses on core computer science concepts including:

* Number systems
* Binary arithmetic
* Data representation

The system is divided into two main modules:

1. Number System Converter
2. Binary Calculator

---

## 🧩 Features

### 🔹 1. Number System Converter

Convert numbers between:

* Binary (Base 2)
* Octal (Base 8)
* Decimal (Base 10)
* Hexadecimal (Base 16)

#### ✅ Functionalities:

* Validate input based on selected base
* Convert from any base → decimal
* Convert from decimal → any base
* Hexadecimal conversion using manual logic

---

### 🔹 2. Binary Calculator

Perform operations on binary numbers.

#### ➕ Arithmetic:

* Binary Addition
* Binary Subtraction

#### 🔄 Complements:

* One’s Complement
* Two’s Complement

#### ✅ Features:

* Binary input validation
* Manual implementation (no built-in functions)
* Menu-driven interface
